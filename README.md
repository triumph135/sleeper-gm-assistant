# sleeper-gm-assistant
Tool to assist with sleeper fantasy football league management - also a newsletter tool that sends a newsletter to everyone in the league with last weeks updates.

- Your first time loading the app;
  - You will need to enter your sleeper leagueID, which can be found in the URL when in your league page on desktop. 
  - You will also need to enter your Sleeper App profile name.
  - To use the advanced featured like trade analysis, weekly snapshot, and chat analysis, you will need to have a claude console account with Tier one API access, and   generate an API key, which will need to be pasted in the Anthorpic API Key field.

 - Add /newsletter to the end of the URL to access the newsletter setup
  - Enter league info including leagueID
  - Then you will need to enter EmailJS public key, senderID, and all of those details
  - Will also need anthropic API Key as well
  - Generate the newsletter
  - Enter the recipient info including team name (generated by pulling team names and profiles via sleeper API), and email address to send to
    - This will all save in your specific browser
  - When done, select the send email button to send the newsletter out.
